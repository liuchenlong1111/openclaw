# AI Prompt 模板

## 基础 Prompt

### 从文本生成题目

```
请根据以下文档内容生成 {count} 道考试题目。

## 文档内容
{content}

## 要求
1. 题目类型包括：单选、多选、不定项、判断、填空、简答
2. 每道题必须包含：题目、答案、解析、难度
3. 单选/多选/不定项题必须提供4个选项
4. 填空题用 __ 表示填空位置
5. 解析必须说明答案在原文中的出处
6. 难度分为：易、中、难

## 输出格式
严格按照以下格式输出：

1.[单选]题目内容？
A. 选项A
B. 选项B
C. 选项C
D. 选项D
答案：A
解析：解析内容
题目难度：中

2.[多选]题目内容？
A. 选项A
B. 选项B
C. 选项C
D. 选项D
答案：BCD
解析：解析内容
题目难度：中

（以此类推）
```

## 按题型生成 Prompt

### 单选题 Prompt

```
请根据文档内容生成 {count} 道单选题。

要求：
1. 每题4个选项，只有1个正确答案
2. 题目考察对文档细节的理解
3. 干扰项要有一定迷惑性但明显错误
4. 包含答案、解析和难度

输出格式：
{序号}.[单选]题目？
A. 选项
B. 选项
C. 选项
D. 选项
答案：X
解析：...
题目难度：X
```

### 多选题 Prompt

```
请根据文档内容生成 {count} 道多选题。

要求：
1. 每题4个选项，有2-4个正确答案
2. 考察对多个知识点的综合理解
3. 包含答案（多个字母连写，如 BCD）、解析和难度

输出格式：
{序号}.[多选]题目？
A. 选项
B. 选项
C. 选项
D. 选项
答案：XXX
解析：...
题目难度：X
```

### 判断题 PROMPT

```
请根据文档内容生成 {count} 道判断题。

要求：
1. 陈述句形式，考察对概念的理解
2. 正确和错误题目比例约为 1:1
3. 错误题目要有明显但不易察觉的错误
4. 包含答案（正确/错误）、解析和难度

输出格式：
{序号}.[判断]陈述内容。
答案：正确/错误
解析：...
题目难度：X
```

### 填空题 PROMPT

```
请根据文档内容生成 {count} 道填空题。

要求：
1. 关键概念、术语、数字适合作为填空
2. 用 __ 表示填空位置
3. 说明填空数和答案形式（完全一致/包含即可）
4. 包含答案、解析和难度

输出格式：
{序号}.[填空]题目内容__。
填空数：N
答案形式：完全一致/包含即可
参考答案1：答案
解析：...
题目难度：X
```

### 简答题 PROMPT

```
请根据文档内容生成 {count} 道简答题。

要求：
1. 考察对概念、原理、流程的理解
2. 答案需要一段文字说明
3. 提供参考答案和评分要点
4. 包含解析和难度

输出格式：
{序号}.[简答]问题？
参考答案：...
解析：...
题目难度：X
```

## 高级 Prompt

### 指定知识点生成

```
请根据文档内容，针对以下知识点生成题目：

知识点：
{topics}

每个知识点生成 {count_per_topic} 道题，类型为 {types}。

文档内容：
{content}
```

### 指定难度生成

```
请根据文档内容生成 {count} 道{difficulty}度题目。

难度说明：
- 易：直接能在原文中找到答案
- 中：需要理解原文，进行简单推理
- 难：需要深入理解，跨段落综合信息

文档内容：
{content}
```
