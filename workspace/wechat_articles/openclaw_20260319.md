## 🌟 什么是 OpenClaw？

OpenClaw 是一个**自托管的 AI 助手网关**，它让你能够在自己掌控的设备上运行 AI 助手，并通过你日常使用的通讯工具（WhatsApp、Telegram、Discord、钉钉等）与它交互。

### 核心理念

- **🔒 数据自主**：所有数据都保存在你自己的设备上
- **⚡ 随时在线**：24/7 待命，像一位真正的助手
- **🔌 多平台支持**：一个 Gateway，连接所有通讯渠道
- **🛠️ 开发者友好**：支持 MCP、技能插件、子代理等高级功能

## 💡 技术亮点

### MCP（Model Context Protocol）支持

OpenClaw 原生支持 MCP 协议，这意味着它可以：
- 调用外部工具和服务
- 与各种 API 无缝集成
- 通过 `mcporter` CLI 管理 MCP 服务器

### 子代理（Sub-agents）

复杂任务？没问题！OpenClaw 可以：
- 生成独立的子代理会话
- 并行处理多个任务
- 通过 ACP（Agent Coding Protocol）运行专业编码代理

### Canvas 实时渲染

在支持的平台上，OpenClaw 可以：
- 渲染实时 UI 界面
- 控制浏览器自动化
- 展示可视化内容
