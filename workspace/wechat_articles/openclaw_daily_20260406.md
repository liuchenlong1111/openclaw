### 2. **whisper-transcribe** —— 本地音视频转录，隐私又方便

基于 OpenAI 开源的 Whisper 模型，现在你可以在本地完成音视频转文字了！

**功能亮点**：
- 🎙️ 支持几乎所有常见音视频格式（mp3, wav, flac, mp4, m4a 等）
- 多种输出格式：纯文本、VTT 字幕、SRT 字幕、JSON、TSV
- 自动检测语言，也可手动指定
- 支持不同模型大小，根据需求平衡速度和准确率
- **完全本地运行**，隐私保护性好，敏感音频放心转

**支持输出格式**：
| 格式 | 用途 |
|------|------|
| txt | 纯文本内容 |
| vtt | Web 播放器字幕 |
| srt | 通用字幕格式 |
| json | 结构化数据处理 |
| tsv | 表格数据分析 |

**中文使用建议**：
- 对准确率要求高：选择 `large` 模型
- 追求速度：选择 `base` 或 `small`
- 平衡方案：`medium` 模型是最佳选择

> 一句话点评：做 podcast、视频号的朋友有福了，自动字幕一键生成！
