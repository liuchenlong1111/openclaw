## 📚 快速上手

### 1. 安装 OpenClaw

```bash
npm install -g openclaw
```

### 2. 启动 Gateway

```bash
openclaw gateway start
```

### 3. 配置通讯渠道

以钉钉为例：

```bash
# 安装钉钉插件
openclaw plugins install @dingtalk-real-ai/dingtalk-connector

# 配置凭证
openclaw config set channels.dingtalk-connector.clientId "your-client-id"
openclaw config set channels.dingtalk-connector.clientSecret "your-secret"

# 重启 Gateway
openclaw gateway restart
```

### 4. 开始对话

打开钉钉，找到你的机器人，发送第一条消息：

> "你好，我是龙龙仔！🐉"

## 🔧 进阶技巧

### 📝 定时任务自动化

使用 `HEARTBEAT.md` + `cron` 实现自动化：

```bash
# 创建每日提醒
openclaw cron add \
  --name "每日早报" \
  --cron "0 9 * * *" \
  --message "早安！今日待办：..." \
  --channel dingtalk
```

### 💾 会话管理

```bash
# 查看活跃会话
openclaw sessions list

# 清理历史会话
openclaw sessions clean
```
